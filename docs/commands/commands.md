<h3 id="cd"> cd </h3>
<p> cd - change directory
      
<blockquote >
cd [options] [path] 
</blockquote >

options:
<br> -L
<br> -P
<br> -e
<br> -@

example:
<blockquote >
cd /home 
<br>cd ~
<br>cd /usr/bin/
<br>cd ..
<br>cd -
</blockquote >

</p>
